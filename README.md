# Vantery AI Orientation Bot Demo

Investor-grade Next.js App Router demo for the Vantery orientation funnel.

## Demo Goals

- Cofounder validation
- Investor wow effect
- Product flow validation
- UX storytelling for conversion

## Product Scope

This demo implements a full local-first orientation pipeline:

1. Orientation landing page
2. Multi-step wizard
3. AI analysis simulation
4. Premium results page
5. University Explorer redirect CTA

No backend is required for this version. Scoring and matching are deterministic and explainable.

## Core Architecture

- App Router routes live in `app/orientation/*`
- Reusable feature components live in `components/orientation/*`
- Domain logic and scoring engine live in `lib/demo-score/*`
- AI explanation generation lives in `lib/ai-explanations/*`
- Program matching lives in `lib/university-matching/*`
- Service contracts and local adapters live in `lib/services/*`
- Local-first state and orchestration live in `hooks/*`
- Mock data and question bank live in `data/*`
- Stable contracts live in `types/*`

## Scoring Model (PRD-Aligned)

- Academic Fit: 35
- Interest Fit: 25
- Career Fit: 20
- Financial Feasibility: 10
- Personality Fit: 10

Total score per major is 100.

## Data Flow

Wizard answers -> scoring helpers -> explainability signals -> university matching -> result snapshot -> premium UI

The UI consumes a single `OrientationResultSnapshot` object generated by the local pipeline service.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Quality Checks

```bash
npm run lint
npm run build
```

## Migration Path (Without Frontend Rewrite)

The current frontend is already structured for adapter swaps:

1. Replace local scoring service with Laravel API scoring endpoint
2. Replace local explanation service with LLM + pgvector retrieval orchestration
3. Replace local mock program matching with real university program tables
4. Keep route structure, UI components, and snapshot contracts unchanged

This keeps the demo production-oriented while preserving fast iteration speed.
